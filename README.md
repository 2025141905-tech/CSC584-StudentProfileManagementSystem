## Project Description

This project is a web-based Student Profile Management System developed using JSP, Servlet, JavaBean, JDBC and Java DB (Derby) following the MVC architecture.

## Implemented Features

1. Add New Student Profile
2. Store Profile Information into Database
3. Display Submitted Profile
4. View All Profiles
5. Search Profile by Student ID or Name
6. Filter Profiles by Programme and Hobbies
7. Form Validation
8. Bootstrap User Interface

## Technologies Used

* HTML
* CSS
* Bootstrap
* JSP
* Servlet
* JavaBean
* JDBC
* Java DB (Derby)
* NetBeans IDE 8.2
* GlassFish Server 4.1.1

## Database

Database Name: StudentProfilesDB

Table Name: Profile

Fields:

* studentID
* name
* programme
* email
* hobbies
* introduction
